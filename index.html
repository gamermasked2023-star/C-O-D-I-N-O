کدینو خط‌هایی که وب را زنده می‌کنند

در کُدینو، هر ایده از یک جرقه کوچک شروع می‌شود و با دقت و خلاقیت تبدیل به تجربه‌ای واقعی روی صفحه نمایش می‌گردد.
اینجا جایی است که طراحی، منطق و جزئیات کنار هم قرار می‌گیرند تا وب نه فقط دیده شود، بلکه حس شود.
هر پروژه با وسواس، هر خط کد با معنا و هر نتیجه با امضای امیرمحمد عبدالهی شکل می‌گیرد؛ جایی که رؤیا و تکنولوژی به هم می‌رسند و نسخه‌ای بهتر از وب ساخته می‌شود.

درباره

HTML زبان پایه‌ی ساخت صفحات وبه. با استفاده از تگ‌ها، ساختار محتوا مثل متن، تصویر و لینک رو مشخص می‌کنه. مرورگر این تگ‌ها رو تفسیر می‌کنه و صفحه رو نمایش می‌ده. می‌شه گفت HTML اسکلت سایت رو می‌سازه و برای زیبایی و تعامل بیشتر به CSS و JavaScript نیاز داره

درباره

CSS یا Cascading Style Sheets زبانیه که برای زیبا کردن و شکل دادن به صفحات وب استفاده می‌شه. در حالی که HTML فقط ساختار و محتوا رو مشخص می‌کنه، CSS ظاهرش رو تعیین می‌کنه؛ مثل رنگ، اندازه فونت، فاصله‌ها، پس‌زمینه و حتی چینش بخش‌های مختلف

درباره

JavaScript یا به اختصار JS زبانیه که به صفحات وب قابلیت تعامل و پویایی می‌ده. اگر HTML ساختار باشه و CSS ظاهر، JavaScript باعث می‌شه صفحه واکنش‌گرا بشه؛ مثلاً وقتی روی دکمه‌ای کلیک می‌کنی چیزی تغییر کنه، فرم‌ها بررسی بشن، یا انیمیشن و افکت اجرا بشه.

درباره

C# یا سی‌شارپ یک زبان برنامه‌نویسی مدرن و قدرتمنده که توسط مایکروسافت ساخته شده. این زبان بیشتر برای ساخت نرم‌افزارهای ویندوز، بازی‌ها (با موتور Unity)، و حتی وب‌اپلیکیشن‌ها استفاده می‌شه. سریع، امن و شیءگراست (یعنی همه‌چیز رو مثل "اشیاء" در نظر می‌گیره تا کدنویسی مرتب‌تر و قابل‌مدیریت‌تر بشه).
